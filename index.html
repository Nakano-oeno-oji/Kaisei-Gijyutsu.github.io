家でpythonの練習をしたい人向けに ~環境構築~

ここでは、家でpythonの練習をしたい人向けに、入れ方とかを書いていきます(一応)

1. python本体の入れ方

まず、公式のpythonダウンロードサイトに飛びます。
そしたら、こんな画面になると思うので、
Download the leatest version for ○○の○○が自分のOSになってることを確認して、
Download python ~バージョン名~ をポチっとします。
ダウンロードが終わったら、エクスプローラーのダウンロードフォルダに行って、python-~バージョン名~-amd64.exe~を開いて、
Add python ~バージョン名~ to pathにチェックを入れて、Customize installationをクリックします。

次は脳死でNextします。もしすべての項目にチェックが入ってなかったら、チェックを入れましょう！

次は、Install all usersにチェックします。ただ、家族と共用のパソコンを使っており、家族とアカウントが分かれている人はチェックしないようにしましょう！！
また、そのあとは階層を低く設定しましょう。C:py/python~バージョン名~とかがいいかなと思います。(なんか打ち間違えてpgとかにしてますね... 恥ずかしい...)

インストールが終わったら、closeを押してOKです！！


2. Atomの入れ方

まず、Atomの公式サイトに飛んで、Downloadのところをポチります。
そのあと、pythonの時と同じようにインストーラーを起動します。
インストールが終わったら自動的にAtomが開きます。そこから説明します。
右上のタブから、settingsを選んで、クリックします。

そしたら、こんな風にsettingsタブが表示されるので、

左下のInstallをクリックして、Install packagesの入力欄に「Japanese-menu」と入力します。

そうしたら、一番上に表示される「Japanese-menu」をインストールします。

これで、環境構築(簡易的ですが)完了です!!
